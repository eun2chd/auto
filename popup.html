토큰 수동 입력

토큰을 직접 입력하거나 추출할 수 있습니다.

자동 추출: Supabase 대시보드나 ERP 시스템 페이지를 열고 "자동 추출" 버튼을 클릭하세요.
수동 입력: 브라우저 개발자 도구(F12) → Network 탭 → 요청 헤더에서 Authorization 값을 복사하세요.
