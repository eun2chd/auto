# ERP 자동 출퇴근 시스템 - Chrome 확장 프로그램

ERP 출퇴근 데이터 관리 및 자동 출퇴근 기능을 제공하는 Chrome 확장 프로그램입니다.

## 설치 방법

1. Chrome 브라우저에서 `chrome://extensions/` 접속
2. 우측 상단의 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 이 프로젝트 폴더 선택

## 아이콘 파일 준비

확장 프로그램을 사용하려면 `icons` 폴더에 다음 크기의 PNG 아이콘 파일이 필요합니다:

- `icon16.png` (16x16 픽셀)
- `icon32.png` (32x32 픽셀)
- `icon48.png` (48x48 픽셀)
- `icon128.png` (128x128 픽셀)

현재 프로젝트에 있는 `erp_icon.ico` 파일을 PNG로 변환하거나, 새로운 아이콘을 준비해주세요.

온라인 변환 도구를 사용하거나, 이미지 편집 프로그램을 사용하여 아이콘을 생성할 수 있습니다.

## 파일 구조

```
ERP-AUTO/
├── manifest.json          # 확장 프로그램 설정 파일
├── popup.html            # 팝업 UI
├── background.js         # 백그라운드 서비스 워커
├── css/
│   └── style.css         # 스타일시트
├── js/
│   └── app.js           # 메인 애플리케이션 로직
├── data/
│   └── vacation-dates.json  # 연차 데이터
└── icons/               # 아이콘 파일들 (준비 필요)
    ├── icon16.png
    ├── icon32.png
    ├── icon48.png
    └── icon128.png
```

## 주요 기능

- 출퇴근 데이터 조회 및 관리
- 자동 출퇴근 기능 (매일 8시 25분)
- 연차 관리
- 결재 시스템 조회
- 인증 토큰 관리

## 사용 방법

1. 확장 프로그램 아이콘을 클릭하여 팝업 열기
2. "출퇴근 데이터 가져오기" 버튼으로 데이터 조회
3. "자동 출퇴근 활성화" 버튼으로 자동 출퇴근 기능 활성화
4. "연차 관리" 버튼으로 연차 날짜 등록

- 주말과 연차 날짜는 자동으로 제외됩니다
